# 倉位控制

* 天才都不一定能有的 100％ 勝率
  * 小賺、大賺、小賠、不大賠
* 入場也很重要



* case：
  * 小賠：入場 -> 止損
  * 小賺：入場 -> 第一止盈點 -> 回到成本價出 (嚴格止盈，否則沒止盈、若回頭，心態更炸)
  * 中賺：入場 -> 第一止盈點  -> 第二止盈點 -> 回到第一止盈出
  * 大賺：入場 -> 第一止盈點  -> 第二止盈點 ->  第三止盈點
* case 有加倉：
  * 中賠：入場 -> 加倉 -> 止損



* do not all in
* N等份法 1,2,4,8,16......
