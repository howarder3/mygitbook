# 繼承

* protected
  * 可繼承 protected member (繼承後，類似 private 的定位)
  * 可以透過 public member func 使用
  * 不可直接使用 member variable
* public
  * 可直接使用 member variable
* private
  * 不繼承
  * 可透過 public member func 使用
  * 不可直接使用 member variable
